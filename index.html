Back end

Back-end é a parte interna de uma aplicação, que não é visível ao usuário, mas é fundamental para o funcionamento do sistema. É o responsável por: Armazenar os dados da aplicação, Garantir que o sistema seja estável e escalável, Processar e armazenar dados de forma rápida e segura

Front end

O front-end é o que seus usuários veem e inclui elementos visuais, como botões, caixas de seleção, gráficos e mensagens de texto. Ele permite que seus usuários interajam com sua aplicação.
